## [1.0.0] - 2026-02-07

### Added

- Copy tab groups to OS clipboard as JSON
- Paste tab groups from clipboard into any Chromium browser
- Save tab groups locally for later restoration
- Restore saved tab groups in current or new window
- Merge multiple tab groups into one with custom name and color
- Split selected tabs from a group into a new group
- Import/export tab groups as JSON files
- View tab groups from all open windows
- Context menu integration for quick save
- Rename and delete saved groups
- Restricted URL filtering (chrome://, file://, etc.)
